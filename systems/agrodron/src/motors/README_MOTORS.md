# Motors component

Компонент `motors` принимает команды движения от `autopilot` / `emergensy` **только через монитор безопасности** и хранит последнее целевое состояние.

Пока интеграция с SITL не определена, поддерживается режим `SITL_MODE=mock`: команда дополнительно публикуется в `SITL_COMMANDS_TOPIC` для наблюдения/подписчиков.
