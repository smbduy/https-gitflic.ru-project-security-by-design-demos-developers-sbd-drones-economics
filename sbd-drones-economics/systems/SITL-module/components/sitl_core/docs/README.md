# SITL Core

Компонент обновления позиций дронов в Redis.

## Описание

Адаптирован из `SITL-module/core.py`. Периодически обновляет координаты движущихся дронов.

## Функционал

- Сканирование ключей `drone:*:state` в Redis
- Обновление координат для дронов в статусе `MOVING`
- Поддержка TTL для состояний

## Конфигурация

- `UPDATE_FREQUENCY_HZ` — частота обновления (по умолчанию 10.0)
- `STATE_TTL_SEC` — TTL состояния (по умолчанию 7200)

## Запуск

```bash
make docker-up
```
