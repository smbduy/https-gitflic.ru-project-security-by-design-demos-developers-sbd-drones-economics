# C4 диаграммы НУС

## Состав

- `C1_Context.puml` — контекст системы (внешние акторы и границы НУС).
- `C2_Containers.puml` — контейнерный уровень НУС.

## Важно

Эксплуатант в диаграммах представлен как внешняя автономная система.
Входные команды (`task.submit`, `task.cancel`, `task.get`) и ответы идут через брокер сообщений (MQTT/Kafka).

## Как открыть

1. Установите VS Code расширение PlantUML.
2. Откройте нужный `.puml` файл.
3. Вызовите команду `PlantUML: Preview Current Diagram`.

## Примечание

Диаграммы используют библиотеку C4-PlantUML по URL include. Если в сети есть ограничения, подключите библиотеку локально и замените `!include` на локальный путь.
